# React Wallet


# Dependencies

[YARN](https://yarnpkg.com/en/docs/install)

# Instructions

```
> git clone https://github.com/fullstackadam/react-wallet.git
> cd react-wallet
> npm install
> yarn run dev-server
```

Go to http://localhost:8080/ in your web browser
